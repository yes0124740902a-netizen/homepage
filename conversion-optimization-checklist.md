# 전환율 최적화 체크리스트 ✅

## 1. 페이지 속도 최적화
- [ ] 이미지 파일 크기 최적화 (WebP 포맷 권장, 100KB 이하)
- [ ] CSS/JS 파일 압축 및 최소화
- [ ] 불필요한 폰트 로딩 제거
- [ ] 브라우저 캐싱 활성화
- [ ] **목표: 페이지 로딩 시간 3초 이내**

## 2. 모바일 최적화
- [ ] 모든 섹션 모바일 반응형 테스트 완료
- [ ] 터치 타겟 크기 최소 48x48px 확보
- [ ] 가로 스크롤 발생하지 않는지 확인
- [ ] 모바일에서 CTA 버튼 명확히 보이는지 확인
- [ ] **목표: 모바일 트래픽 전환율 PC 대비 80% 이상**

## 3. CTA (Call-to-Action) 최적화
- [ ] 스크롤 없이 보이는 영역에 CTA 배치 (Above the Fold)
- [ ] 페이지 전체에 최소 3개 이상의 CTA 버튼 배치
- [ ] CTA 버튼 색상: 고대비 색상 사용 (예: 빨강, 주황)
- [ ] 액션 지향적 문구 사용 ("지금 시작하기", "무료로 체험하기")
- [ ] 버튼 크기: 충분히 크고 눈에 띄게

## 4. 신뢰성 구축
- [ ] 고객 후기/리뷰 최소 3개 이상 표시
- [ ] 실제 고객 이름 및 사진 포함 (가능한 경우)
- [ ] 별점 또는 평점 시각화
- [ ] 수강생 수, 만족도 등 숫자로 증명
- [ ] 환불 정책 명시로 구매 불안 제거

## 5. 폼 최적화
- [ ] 입력 필드 개수: 3~5개 이내로 제한
- [ ] 필수/선택 항목 명확히 표시
- [ ] 자동완성 기능 활성화
- [ ] 에러 메시지 친절하게 작성
- [ ] 제출 후 확인 메시지 표시

## 6. 카피라이팅
- [ ] 헤드라인: 구체적 혜택 명시 (예: "하루만에 웹사이트 완성")
- [ ] 숫자 활용 (예: "500만원 절감", "3,000명+ 수강생")
- [ ] 긴급성 부여 (예: "이번 달 한정", "선착순 20명")
- [ ] "당신"보다 "귀하", "여러분" 등 친근한 톤
- [ ] 전문 용어 최소화, 쉬운 단어 사용

## 7. 시각적 요소
- [ ] 히어로 섹션에 매력적인 이미지 또는 동영상
- [ ] 섹션별 아이콘/이모지로 시각적 구분
- [ ] 충분한 여백(White Space) 확보
- [ ] 브랜드 컬러 일관성 유지
- [ ] 폰트 크기: 본문 16px 이상, 헤드라인 24px 이상

## 8. SEO 기본 설정
- [ ] 페이지 타이틀 태그 최적화 (60자 이내)
- [ ] 메타 디스크립션 작성 (155자 이내)
- [ ] 헤딩 태그 계층 구조 (H1 → H2 → H3)
- [ ] 이미지 alt 텍스트 추가
- [ ] Open Graph 태그 설정 (소셜 공유용)

## 9. 사용자 경험 (UX)
- [ ] 스크롤 깊이: 각 섹션 논리적 순서로 배치
- [ ] 네비게이션 메뉴 (필요 시): 명확하고 간결하게
- [ ] 404 에러 페이지 없는지 확인
- [ ] 링크 클릭 시 예상 결과 일치
- [ ] 팝업 사용 최소화 (사용 시 쉽게 닫을 수 있게)

## 10. A/B 테스트 계획
- [ ] 테스트할 요소 선정 (헤드라인, CTA 버튼 색상 등)
- [ ] 구글 애널리틱스 또는 Hotjar 설치
- [ ] 전환율 목표 설정 (예: 방문자 대비 신청률 5%)
- [ ] 2주 단위로 데이터 분석 및 개선
- [ ] 사용자 피드백 수집 채널 마련

## 우선순위 액션 아이템 (즉시 실행)

### 🔥 High Priority
1. **모바일 반응형 테스트** - 모바일 트래픽이 전체의 60% 이상
2. **페이지 로딩 속도** - 3초 이상 시 이탈률 40% 증가
3. **CTA 버튼 가시성** - 전환율에 직접적인 영향

### ⚡ Medium Priority
4. **고객 후기 추가** - 신뢰도 30% 향상
5. **카피라이팅 개선** - 헤드라인과 서브헤드라인 A/B 테스트
6. **폼 간소화** - 입력 필드 1개 감소 시 전환율 10% 증가

### 📊 Low Priority (장기)
7. **SEO 최적화** - 장기적 트래픽 증대
8. **A/B 테스트 설정** - 지속적 개선
9. **고급 애널리틱스** - 사용자 행동 패턴 분석

## 성과 측정 지표 (KPI)

| 지표 | 목표값 | 측정 방법 |
|------|--------|-----------|
| 페이지 로딩 시간 | 3초 이내 | Google PageSpeed Insights |
| 이탈률 (Bounce Rate) | 40% 이하 | Google Analytics |
| 평균 체류 시간 | 2분 이상 | Google Analytics |
| 전환율 (Conversion Rate) | 5% 이상 | 신청 수 / 방문자 수 |
| 모바일 전환율 | PC 대비 80% | Google Analytics |
